公式步骤

一、 初始设定与准备

1. 指定输入参数并初始化矩阵

在宏观材料属性与微观近场动力学参数之间建立映射。

核心是计算键常数（Micro-modulus / Bond Constant）。

PD 的应变能密度与经典弹性力学的应变能密度相等，可以反推出键常数 c

  • 公式（二维平面应力）：

    c=9Eπhδ3

  • 参数：

    • c : 键常数（单位体积的刚度）
    • E : 材料的宏观杨氏模量（抵抗变形的物理量）
    • h : 板的厚度
    • δ : 近场范围（Horizon）半径

2. 确定时域积分的稳定时间步长

为了保证显式积分不发散，时间步长必须满足 CFL (Courant-Friedrichs-Lewy) 稳定性条件。

  • 公式（显式动力学临界步长）：

    Δt≤2ρ∑jc|ξij|Vj

    在常规二维网格中，通常采用一个安全的估算值：

    Δt=0.82ρΔxπδ2hc

  • 参数：

    • Δt : 时间步长
    • ρ : 材料密度
    • Δx : 质点间距

  • (注：若使用 ADR 求解静态问题，时间为虚拟时间，通常直接设 Δt=1.0 )

二、 空间离散与近场关系构建

3. 生成质点

将连续体离散为 N 个质点，每个质点代表一个子域的体积。

  • 公式：

    Vi=Δx⋅Δy⋅h

  • 参数：

    • Vi : 质点 i 的体积

4. 寻找并存储近场族成员

确定拓扑连接关系。

  • 公式（近场判定准则）：

    |ξij|=|xj−xi|≤δ

  • 参数：

    • xi,xj : 质点 i 和 j 的初始空间坐标向量
    • ξij : 初始相对位置向量（初始键长）

5. 预置裂纹（可选）

通过引入历史相关标量来截断初始相交的键。

  • 公式：

    μ(0,ξij)=0(如果键 ξij 与预制裂纹面相交)

  • 参数：

    • μ : 键的历史依赖状态参数（1 表示完好，0 表示断裂）

6. 计算表面修正因子

为了补偿边界处缺失导致的“表面效应”软化。

  • 公式（应变能等效法）：

    λi=WanalyticalWPD,i=916Eε212∑j(12cε2|ξij|)Vjfvol

  • 参数：

    • λi : 质点 i 的表面修正因子（用于放大边界点的刚度）
    • ε : 施加的虚拟微小拉伸应变
    • fvol : 体积修正系数（处理边界相交的质点体积占比，取值范围 0 到 1）

三、 运动学边界与条件初始化

7. 施加初始条件

  • 公式：

    ui(0)=0,vi(0)=vinitial

  • 参数：

    • ui,vi : 位移与速度向量

8. 构建稳定的质量矩阵（仅限 ADR）

为保证虚拟动态过程绝对稳定而人工设定的虚拟质量。

  • 公式：

    mi≥14Δt2∑j|Kij|≈0.25Δt2(πδ2h)cΔx⋅safety

  • 参数：

    • mi : 质点 i 的虚拟质量对角元
    • safety : 安全系数（通常取 1.1 到 5.0）

四、 时域积分循环（核心计算阶段）

9. 开始时域积分

时间步推进 tn+1=tn+Δt 。

10. 施加边界条件

对于处于边界层的质点（如算例中的上下 3 层质点），强制赋予当前时间的位移或速度。

11. 计算 PD 相互作用力的总和

这是最核心的本构力计算。

  • 变形后的相对位置（键的当前状态）：

    yij=ξij+ηij=(xj−xi)+(uj−ui)

  • 键的伸长率（Stretch）：

    s=|yij|−|ξij||ξij|

  • 单根键的相互作用力密度向量：

    fij=c⋅s⋅μ⋅λscr⋅yij|yij|

  • 质点 i 受到的 PD 总内力：

    Li=∑jfijVjfvol

  • 参数：

    • λscr : 考虑了点 i 和 j 表面修正因子的综合修正系数（通常用椭圆公式插值）

12. 进行损伤预测（断裂更新）

判断键是否达到极限，并更新宏观损伤。

  • 公式（临界伸长率与断裂）：

    s0=4πGc9Eδ 如果 s>s0, 则 μ(t,ξij)=0

  • 公式（宏观损伤度）：

    Di=1−∑jμ(t,ξij)Vjfvol∑jVjfvol

  • 参数：

    • s0 : 临界伸长率
    • Gc : 材料的断裂能（Fracture Energy）
    • Di : 质点 i 的损伤度（0 为完好，1 为完全破坏）

13. 计算自适应动力松弛参数（仅限 ADR）

根据系统不平衡力动态调整阻尼，吸收动能。

  • 公式：

    cn=2∑iuiT⋅(Linmi−Lin−1mi)/(Δt⋅vin−1/2)∑iuiTui

  • 参数：

    • cn : 当前时间步的全局最优阻尼系数

14. 执行时域积分得到位移和速度

根据所选的物理模型更新运动学状态。

  • 路径 A：显式动力学（真实物理时间，模拟动态断裂）

    ain=Lin+binρ vin+1/2=vin−1/2+ainΔt uin+1=uin+vin+1/2Δt

  • 路径 B：自适应动力松弛 ADR（虚拟时间，求解静态平衡）

    vin+1/2=(2−cnΔt)vin−1/2+2Δt(Lin+bin)/mi2+cnΔt uin+1=uin+vin+1/2Δt

  • 参数：

    • ain : 质点加速度
    • bin : 外加体力（如重力）

预处理与离散化 (Pre-processing & Discretization)

几何离散 (Geometry Discretization)：

将连续体划分为有限数量的“物质点” (Material Points)。确定每个点的初始坐标 ( x )、所占体积 ( V ) 和材料密度 ( ρ )。

设定材料参数：

定义体积模量 ( K )、剪切模量 ( G )、临界伸长率 ( s0 或 β0 ) 等宏观材料属性，并将其转化为 PD 本构模型中的微观参数（如微观模量 c ）。

施加初始条件：

设定物体在 t=0 时刻的初始速度和初始位移（通常为 0）。

初始化设定 (Initialization)

在时间步循环开始前，需要为系统建立初始的“拓扑关系”。

  1. 邻域搜索 (Neighbor Search)： 设定视界半径 (Horizon, δ )。遍历所有物质点，找出距离在 δ 范围内的所有邻居点，建立该点的“族” (Family)。
  2. 建立初始键 (Initial Bonds)： 记录点与邻居点之间的初始相对位置矢量 ( ξ=xj−xi ) 和初始距离。

核心求解循环 (Time Integration Loop)

在每一个时间步 t 内，执行以下操作：

  1. 运动学更新： 根据当前各点的新位置，计算每一个“键”当前的相对位移 ( η ) 和伸长率 ( s )。
  2. 损伤与断键判定 (Damage Assessment)： * 检查伸长率是否超过临界值。如果超过，该键发生不可逆断裂（状态标记为 0），反之完好（状态标记为 1）。
    • 计算并更新每个物质点的宏观损伤度 ( ϕ )。
  3. 本构力计算 (Constitutive Force Calculation)： * 根据选择的本构模型（如键基 Bond-based、态基 State-based），结合当前键的变形和未断裂状态，计算每个邻居点对该点施加的相互作用力（成对力函数 f ）。
    • 将所有邻居施加的力进行体积积分（在离散模型中为求和），得到该点受到的总内力。
  4. 施加边界条件与外力： 将外部施加的体力（如重力）、面力（转化为局部体积力）或强制位移（如施加在边界层物质点上的拉伸）加入计算。
  5. 求解运动方程 (Solving Equation of Motion)： * 将总力代入牛顿第二定律，得出加速度。
    • 使用时间积分算法（如中心差分法/Velocity Verlet），根据加速度更新出下一时刻 ( t+Δt ) 该点的速度和位置。

后处理与可视化 (Post-processing)

计算完成后（或者在循环中按设定的频率），将数据输出以供分析。

  1. 数据导出： 将各个时间步下所有物质点的坐标、位移分量、速度以及损伤度标量等数据，导出为特定格式的文件（如 .dat、.csv 或 .plt）。
  2. 可视化渲染： 将导出的数据序列导入到专门的后处理软件（如 Tecplot 或 Paraview）中。通过对“损伤度”字段进行云图映射（例如 0 为蓝色完好，1 为红色断裂），就可以直观地观察到裂纹的萌生、分叉和动态扩展过程。
